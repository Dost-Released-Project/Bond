# **기획 의도 기준**

본 기획서의 모든 시스템은 다음 세 가지 기획 의도에 부합하도록 설계되어야 합니다. 각 시스템을 구현하거나 검토할 때 아래 기준을 참고해 주세요.

| \# | 기획 의도 | 시스템적 구현 방향 |
| :---- | :---- | :---- |
| 1 | 플레이어 서사 중심에서 벗어나 다양한 캐릭터들 사이의 서사를 제공합니다. | 리더는 관리자일 뿐이며, 대원 간의 관계 변화가 게임플레이에 직접 영향을 미쳐야 합니다. |
| 2 | 캐릭터의 성격을 텍스트가 아닌 실제 리액션 로직과 행동으로 데이터화하여 제공합니다. | 성향(Trait)은 설명 텍스트가 아닌, 전투에서 실제로 발동하는 행동 데이터로 구현됩니다. |
| 3 | 플레이어가 초기에는 지적 쾌감을, 후기에는 스스로 쌓아올린 서사를 느끼게 합니다. | 리액션 로직 설계의 재미와, 누적된 파티 관계 데이터가 서사로 전환되는 구조를 만들어야 합니다. |

**✦  구현 체크리스트 — 사운드 및 VFX**

# **사운드 및 VFX**

*기획 의도: "내 작전이 먹히고 있는가?"와 "누가 통제를 잃어가고 있는가?"를 설명 없이 감각적으로 전달하는 것이 목표입니다.*

## **1\. 디자인 철학**

* **설계의 가시화:** 유저가 짠 리액션 로직이 '연쇄'되는 과정을 리듬감 있게 전달합니다.

* **불협화음의 강조:** 리더의 통제를 벗어난 '돌발 행동'은 불쾌하고 강렬한 감각으로 피드백합니다.

* **저채도의 세계관:** 미개척지의 황량함을 표현하기 위해 전체적으로 채도를 낮추되, 이펙트의 빛과 소리만은 선명하게 강조합니다.

## **2\. VFX 세부 가이드**

| 분류 | 연출 상황 | 핵심 연출 방식 |
| :---- | :---- | :---- |
| 리액션 체인 | 설계된 연쇄 발동 시 | 캐릭터 간 실선(Connection Line) 연결, 타격 시 스파크 강조 |
| 돌발 행동 | 지능 판정 실패/본능 발현 | 화면 노이즈(Glitch), 붉은색 섬광, 캐릭터 실루엣 왜곡 |
| 성향 각성 | 긍정적 성향 발현 시 | 캐릭터 발밑에 고유 문양 이펙트, 백색 발광 효과 |
| 보급 완료 | 본국 물자 도착 시 | 화면 상단에서 하단으로 떨어지는 강한 빛의 기둥 연출 |

## **3\. 사운드 세부 가이드**

| 분류 | 사운드 성격 | 세부 연출 내용 |
| :---- | :---- | :---- |
| 리액션 SFX | 리드미컬/금속성 | 연쇄가 이어질수록 사운드 피치(Pitch)가 점진적으로 상승 |
| 돌발 행동 SFX | 불협화음/기괴함 | 낮은 저음의 불쾌한 잔향, 깨지는 듯한 디지털 노이즈음 |
| 캐릭터 보이스 | 현장감/처절함 | 연쇄 시 "연결해\!", "다음\!" 등 상호작용 대사 출력 |
| 환경 앰비언스 | 고립감/긴장감 | 바람 소리, 멀리서 들리는 괴생명체 울음, 심장 박동음 |

## **4\. 연출 우선순위**

| 우선순위 | 상황 | 연출 강도 |
| :---- | :---- | :---- |
| 최우선 | 돌발 행동 (명령 무시) | 가장 크고 불쾌한 연출. 설계가 깨진 순간을 즉각 인지시킵니다. |
| 중간 | 리액션 연쇄 성공 | 경쾌하고 연결감 있는 사운드. 설계 성공의 쾌감을 전달합니다. |
| 일반 | 리더 직접 공격, 이동, 빗맞음 | 이펙트 크기를 줄여 화면 시인성을 확보합니다. |

## **5\. 시스템 내러티브 연출**

* **리더의 일지:** 종이가 거칠게 넘어가는 소리, 펜촉이 종이를 긁는 사운드를 통해 '기록'의 무게감을 전달합니다.

* **본국 보급:** 차갑고 기계적인 기계음(통신 노이즈)을 배치하여, '철저한 비즈니스 파트너'로서의 국가 이미지를 강화합니다.

## **6\. 기술적 주의사항**

* **VFX 시인성:** 4인 연쇄가 동시에 터질 때 화면 전체가 가려지지 않도록 이펙트의 중심부는 투명하게 처리하고 외곽선 위주로 강조합니다.

* **사운드 채널:** 연쇄 사운드가 겹칠 때 소리가 찢어지지 않도록 리미터를 설정하고, 연속된 타격음은 각각 미세하게 피치를 다르게 하여 지루함을 방지합니다.
