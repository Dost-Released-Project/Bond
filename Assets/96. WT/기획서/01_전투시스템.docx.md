# **전투 시스템**

*기획 의도: 기획 의도 2번(성격의 데이터화)과 3번(지적 쾌감 → 서사)의 핵심 구현 공간입니다. 캐릭터의 성향은 전투 중 리액션 로직을 통해 행동으로 증명되며, 플레이어는 이 로직을 설계하고 관찰하는 과정에서 지적 쾌감을 얻습니다.*

## **1\. 턴 구조**

### **선제권 결정 및 고정 순회**

* **선제권:** 전투 시작 시 단 한 번, 모든 참여자(최대 8명)의 민첩(AGI) 스탯을 비교하여 1\~8번의 행동 순서를 확정합니다.

* 0.001 단위 소수점 판정 적용으로 동점 방지

* **무한 루프:** 확정된 순서는 전투 종료 시까지 1 → 2 → ... → 8 → 1 순서로 고정되어 반복됩니다.

### **전방위 리액션 개입**

* **발동 주체:** 현재 행동 중인 모든 캐릭터(액터)의 활동이 트리거가 됩니다.

* **개입 로직:** 특정 캐릭터의 턴 진행 중이라도, 해당 캐릭터의 행동이 다른 동료의 리액션 조건을 만족하면 즉시 개입이 발생합니다.

* **복귀 로직:** 모든 연쇄 리액션이 종료되면 타임라인은 중단되었던 원래 캐릭터의 순번으로 복귀합니다.

* **우선순위:** 여러 동료가 동시에 리액션 조건을 만족할 경우, 민첩(AGI) 수치가 높은 캐릭터부터 순차적으로 실행됩니다.

## **2\. 리액션 데이터 체계**

리액션은 **\[관찰 대상(Target)\] \+ \[트리거(Trigger)\] \+ \[실행 스킬(Action)\]**의 결합으로 완성됩니다.

### **데이터 항목 정의**

| 필드명 | 설명 |
| :---- | :---- |
| ID | 트리거 자체의 고유 식별 번호 |
| Category / SubCategory | 로직 그룹핑 및 UI 필터링을 위한 분류 체계 |
| Trigger\_Key | 코드상에서 이벤트 리스너가 감시할 고유 문자열 식별자 |
| Description | 해당 트리거가 발생하는 구체적인 상황에 대한 기획적 설명 |
| Value\_Param | 스탯 기반 트리거 등에서 사용하는 가변 수치값 (예: HP N% 미만) |

### **트리거 조건 분류**

| 분류 | 키 예시 | 설명 |  |
| :---- | :---- | :---- | :---- |
| 공격 기반 (OFFENSIVE) | TRG\_OFF\_CRIT, TRG\_OFF\_KILL | 치명타 적중, 적 처치, 특정 속성 사용 등 |  |
| 방어/피격 기반 (DEFENSIVE) | TRG\_DEF\_TG\_IN | 피격 예고, 아군 피격, 회피 성공, 상태이상 발생 등 |  |
| 전장 상황 기반 (SITUATIONAL) | TRG\_SIT\_MISS | 공격 빗나감, 적 스킬 캐스팅, 구역 진입/이탈 등 |  |
| 스탯 기반 (STAT\_BASED) | TRG\_STA\_HP\_L | 개별/파티 평균 체력 저하, 자원(MP) 고갈 등 |  |

## **3\. 핵심 스탯**

| 스탯 | 주요 역할 |
| :---- | :---- |
| 힘 (STR) | 최대 체력(HP), 방어도(데미지 감소량), 물리 기반 스킬 위력 결정 |
| 민첩 (AGI) | 선제권 결정, 리액션 쿨다운 가속 (민첩 \+50당 2라운드마다 쿨다운 1 추가 감소), 회피/명중 보너스, 리액션 우선순위 |
| 지능 (INT) | 주문 기반 스킬 위력, 돌발 행동(성향) 억제력 결정 |

## **4\. 돌발 행동 확률 및 유대적 각성**

**공식:** 최종 발동 확률 \= \[기본 확률 \+ 스트레스 가산치\] \- (지능 × 계수 \+ 파티 관계 보너스)

| 지능 수치 | 효과 |
| :---- | :---- |
| 지능 0 (표준) | 성향의 기본 발동률 유지 |
| 지능 100 (한계치) | 발동 확률을 최저치(5%)까지 강제 억제 |
| 지능 \-50 (저지능) | 성향 발동률이 증폭되어 통제가 매우 어려워짐 |

* **최저 확률 보장:** 성향의 변수를 위해 최소 5%의 발동 확률은 하한선으로 고정됩니다.

* **유대적 각성 (Bond Awakening):** 스트레스 임계치에서 돌발 행동 판정(실패) 시, 관계 수치에 따라 확률적으로 발동하여 스트레스 정화 및 행동 효과를 비약적으로 증가시킵니다.

*기획 의도: 돌발 행동은 단순한 패널티가 아닙니다. 위기 상황에서 캐릭터의 성향이 행동으로 드러나는 순간(의도 2번)이며, 유대적 각성으로 전환될 때 플레이어가 쌓아온 관계의 서사가 보상으로 돌아옵니다(의도 3번).*

## **5\. 파티 관계 및 스트레스**

| 구분 | 요인 |
| :---- | :---- |
| 관계 상승 | 리액션 성공, 위기 구호(HP 30% 이하 보호), 연쇄 처치, 전투 외 상호작용 |
| 스트레스 상승 | 체력 위기(30% 이하), 동료 사망(파티 전체), 연속 피격, 특수 디버프 발생 |
