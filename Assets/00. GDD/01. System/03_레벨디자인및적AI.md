# **기획 의도 기준**

본 기획서의 모든 시스템은 다음 세 가지 기획 의도에 부합하도록 설계되어야 합니다. 각 시스템을 구현하거나 검토할 때 아래 기준을 참고해 주세요.

| \# | 기획 의도 | 시스템적 구현 방향 |
| :---- | :---- | :---- |
| 1 | 플레이어 서사 중심에서 벗어나 다양한 캐릭터들 사이의 서사를 제공합니다. | 리더는 관리자일 뿐이며, 대원 간의 관계 변화가 게임플레이에 직접 영향을 미쳐야 합니다. |
| 2 | 캐릭터의 성격을 텍스트가 아닌 실제 리액션 로직과 행동으로 데이터화하여 제공합니다. | 성향(Trait)은 설명 텍스트가 아닌, 전투에서 실제로 발동하는 행동 데이터로 구현됩니다. |
| 3 | 플레이어가 초기에는 지적 쾌감을, 후기에는 스스로 쌓아올린 서사를 느끼게 합니다. | 리액션 로직 설계의 재미와, 누적된 파티 관계 데이터가 서사로 전환되는 구조를 만들어야 합니다. |

| 완료 | 구현 항목 | Input | Output | 참고 |

| ☐ | **IsActionable() 체크** | Stun / Silence / Bind 플래그 상태 | 리액션 실행 스택 생성 여부 (true/false) | 1 |
| ☐ | **스트레스 집중 타격 로직** | Skill.Stress\_Power \+ Target.INT\_Coefficient | Target.Stress 증가량 | 1 |
| ☐ | **AI 타겟팅 가중치** | 파티 전체 INT \+ 스트레스 수치 | Focus Fire 우선 타겟 선정 | 1 |
| ☐ | **Knockback/Pull 전술적 고립** | 스킬 사용 신호 \+ 타겟 위치 | Transform.Position 변경값 | 1 |
| ☐ | **연막/암흑 Raycast 비활성화** | 연막/암흑 상태이상 적용 신호 | 트리거 감지 Raycast 활성화 여부 (false) | 1 |

# **레벨 디자인 및 적 AI**

*기획 의도: 적 AI는 기획 의도 3번(지적 쾌감)의 검증 공간입니다. 적은 플레이어가 설계한 리액션 체인을 능동적으로 방해하며, 플레이어는 이 방해를 극복하는 과정에서 설계의 가치를 확인합니다.*

## **1\. 적 AI 전술 체계**

### **\[전술 1\] 행동 불능(CC) 연계: 리액션 발동 원천 봉쇄**

* **기획 의도:** 트리거 조건을 만족해도 캐릭터가 행동 불가 상태면 리액션 개입을 차단하여 연쇄를 끊습니다.

* **구현 로직:** 리액션 이벤트 리스너 호출 시 IsActionable() 상태를 최우선 체크. Stun/Silence/Bind 플래그가 True일 경우 리액션 실행 스택 생성하지 않고 무시 처리

* **AI 우선순위:** 현재 리액션 쿨다운이 완료(Cooldown \== 0)된 아군을 탐색하여 최우선적으로 상태이상 스킬 투사

### **\[전술 2\] 스트레스 집중 타격: 리액션 실패(돌발 행동) 유도**

* **기획 의도:** 데미지보다 스트레스를 높여 지능(INT) 판정 실패를 유도하고, 아군이 스스로 설계를 무너뜨리게 만듭니다.

* **구현 로직:** Target.Stress \+= Skill.Stress\_Power \* (1.0 \- Target.INT\_Coefficient) / 스트레스 수치 임계치 이상인 타겟에게 Targeting\_Weight 가중치 부여 (30%\~50%)

* **AI 우선순위:** 파티 내 INT가 가장 낮거나 현재 스트레스 수치가 높은 아군을 점사(Focus Fire)

### **\[전술 3\] 전술적 고립: 트리거 인식 제한**

* **기획 의도:** 아군 간의 거리나 시야를 조작하여 리액션 트리거 인식 범위(Target\_Scope)를 차단합니다.

* **구현 로직:** Knockback/Pull 스킬을 통해 Transform.Position 변경. '연막/암흑' 상태이상 시 트리거 감지 레이캐스트(Raycast) 일시 비활성화

* **AI 우선순위:** 리액션 연쇄의 중심에 있는 캐릭터(연결 고리)를 물리적으로 격리

## **2\. 적 유닛 프로파일**

| 유닛명 | 유형 | 전술 역할 |
| :---- | :---- | :---- |
| 전열 가디언 | REACTION\_BAITER | 높은 방어력으로 아군의 공격 트리거를 강제 소모하게 유도하여 리액션 쿨다운을 유발 |
| 후열 마인드 브레이커 | STRESS\_MINER | 스트레스 증폭 특화. 지능(INT)이 낮은 아군을 집중 공격 |
| 지원 디스럽터 | CC\_SPECIALIST | 리액션 대기 중인 아군에게 침묵/기절을 걸어 체인을 직접 차단 |
| 보스 오버로드 | PHASE\_CHANGE\_BOSS | 유대 수치가 높은 아군 쌍에게 페널티를 부여하며, 페이즈 전환 시 더 정교한 리액션 차단 패턴 사용 |

## **3\. 적 데이터 시트 항목 (CSV/DB)**

| 필드명 | 타입 | 설명 |
| :---- | :---- | :---- |
| AI\_Preference\_Weight | Float | 특정 타겟(저지능, 리액션 대기자 등) 선호도 가중치 수치 |
| Interrupt\_Priority | Bool | 아군의 리액션 체인 도중 적이 강제 개입할 수 있는지 여부 |
| Stress\_Value | Float | 스킬 적중 시 가산할 기본 스트레스량 |

## **4\. 전장 기믹 및 존 시스템**

| 존 이름 | 효과 및 구현 |
| :---- | :---- |
| 침묵의 성역 (Silence Zone) | 구역 진입 시 Action\_Blocker 컴포넌트 활성화 → IsActionable 강제 False |
| 공포의 근원 (Stress Object) | 턴 시작 시 범위 내 아군의 Stress\_Point 상시 가산 |

* **Phase Trigger:** 보스의 HP 혹은 특정 라운드 도달 시 AI\_Logic\_Type을 교체하여 타겟팅 알고리즘을 고도화합니다.
