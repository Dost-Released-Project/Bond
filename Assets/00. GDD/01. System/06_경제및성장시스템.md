# **기획 의도 기준**

본 기획서의 모든 시스템은 다음 세 가지 기획 의도에 부합하도록 설계되어야 합니다. 각 시스템을 구현하거나 검토할 때 아래 기준을 참고해 주세요.

| \# | 기획 의도 | 시스템적 구현 방향 |
| :---- | :---- | :---- |
| 1 | 플레이어 서사 중심에서 벗어나 다양한 캐릭터들 사이의 서사를 제공합니다. | 리더는 관리자일 뿐이며, 대원 간의 관계 변화가 게임플레이에 직접 영향을 미쳐야 합니다. |
| 2 | 캐릭터의 성격을 텍스트가 아닌 실제 리액션 로직과 행동으로 데이터화하여 제공합니다. | 성향(Trait)은 설명 텍스트가 아닌, 전투에서 실제로 발동하는 행동 데이터로 구현됩니다. |
| 3 | 플레이어가 초기에는 지적 쾌감을, 후기에는 스스로 쌓아올린 서사를 느끼게 합니다. | 리액션 로직 설계의 재미와, 누적된 파티 관계 데이터가 서사로 전환되는 구조를 만들어야 합니다. |

**✦  구현 체크리스트 — 경제 및 성장 시스템**

### **자원 체계**

| 완료 | 구현 항목 | Input | Output | 참고 |
| :---: | ----- | ----- | ----- | ----- |
| ☐ | **현지 자원 획득 로직** | 전투 승리 / 유적 조사 / 적 분석 이벤트 | 개척 데이터 수량 증가 | 1 |
| ☐ | **외부 자원 요청 로직** | 유저의 보급 요청 선택 \+ 기여도 수량 | 기여도 차감 \+ 배급품 / 증원 / 각성제 지급 | 1 |
| ☐ | **기여도 소모 → 보급 결제** | 보급 요청 타입 \+ 기여도 수량 | 기여도 차감 \+ 해당 보급 지급 처리 | 3 |

### **성장 시스템**

| 완료 | 구현 항목 | Input | Output | 참고 |
| :---: | ----- | ----- | ----- | ----- |
| ☐ | **개척 데이터 소비 → 숙련도 상승** | 소비할 개척 데이터 수량 | 스탯 가산치 \+ 해금된 스킬 ID 목록 | 2 |
| ☐ | **정신 각성제 → 성향 슬롯 확장** | 각성제 소비 신호 \+ 대상 캐릭터 | 캐릭터 성향 슬롯 수 \+1 | 2 |
| ☐ | **증원 인원 무작위 성향 생성** | 증원 요청 신호 | 무작위 Trait\_ID 배정된 신규 대원 객체 | 1 |

# **경제 및 성장 시스템**

*기획 의도: 경제 시스템은 기획 의도 1번(캐릭터 간 서사)을 압박감으로 구현하는 장치입니다. 한정된 자원과 관리 딜레마가 파티원들 사이의 갈등과 선택을 강요하며, 이 선택들이 누적되어 플레이어만의 서사를 만들어냅니다.*

## **1\. 자원 체계**

### **현지 자급자족 자원 (Field Resources)**

| 자원명 | 주요 획득처 | 주요 소비처 및 기능 |
| :---- | :---- | :---- |
| 개척 데이터 (Log) | 전투 승리, 유적 조사, 적 분석 | 직업 숙련도 강화: 기초 스탯 상승 및 신규 스킬 해금 |
| 야전 소모품 | 탐사 채집, 오브젝트 파괴 | 캠프 정비: 응급 처치 및 요리 (생존 및 스트레스 미량 회복) |

### **외부 의존 자원 (State Supplies)**

| 자원명 | 주요 획득처 | 주요 소비처 및 기능 |
| :---- | :---- | :---- |
| 개척 기여도 | 성과 보고 (지도 완성, 거점 확보) | 보급 결제 수단: 모든 외부 물자 및 인원 요청 시 소모 |
| 왕실 배급품 | 본국 보급 요청 | 이성 유지: 스트레스 대폭 감소 및 부정 성향 발동 억제 |
| 증원 인력 | 본국 증원 요청 | 인원 보충: 손실 대원 대체 (무작위 성향의 신규 대원 합류) |
| 정신 각성제 | 본국 희귀 보급 | 정체성 확장: 캐릭터의 성향 슬롯 추가 확장 |

## **2\. 성장 시스템**

### **직업 숙련도 (성장)**

현지에서 얻은 개척 데이터를 소비하여 물리적인 체급을 키우는 과정입니다.

* **효과:** 힘, 민첩, 지능 등 기초 스탯이 상승합니다.

* **기능:** 리더의 작전 명령에 매칭할 수 있는 스킬 라이브러리가 확장됩니다.

### **성향 슬롯 확장 (발현)**

캐릭터가 극한 상황에서 드러내는 인격의 복합성을 의미하며, 정신 각성제나 특정 사건을 통해서만 개방됩니다.

* **의미:** 슬롯이 늘어날수록 캐릭터는 더 많은 상황(트리거)에 반응하는 입체적인 인물이 됩니다.

* **기능:** 리더는 확장된 슬롯에 새로운 성향을 채워 넣어 더욱 정교하고 복잡한 리액션 체인을 설계할 수 있습니다.

## **3\. 경제 루프 구조**

| 단계 | 내용 |
| :---- | :---- |
| 1\. 성과 확보 | 탐사를 통해 본국에 보고할 기여도와 숙련에 필요한 데이터를 수집 |
| 2\. 본국 보고 | 리더의 인장으로 성과를 송신, 배급품/증원 인력/각성제 중 요청 |
| 3\. 정비 및 강화 | 데이터로 대원 체급 강화, 신규 인원 성향 확인, 에이스에게 각성제 투여 |
| 4\. 전술 재설계 | 변화된 인적 구성과 확장된 슬롯에 맞춰 리액션 지침 업데이트 |

## **4\. 관리 리스크와 딜레마**

대원은 현지 조달이 불가능한 귀한 자산입니다. 대원을 잃는 것은 그간 투자한 숙련도와 각성제의 영구적 손실을 의미합니다.

| 선택 | 내용 |
| :---- | :---- |
| 배급품 | 현재 인원의 정신을 케어하여 성향 돌발 행동을 억제 |
| 증원 | 죽은 자를 대신할 새 피를 수혈 (무작위 성향의 신규 대원) |
| 각성제 | 에이스의 성향 슬롯을 확장하여 한계를 돌파 |
