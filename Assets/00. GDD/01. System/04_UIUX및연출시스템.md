# **UI/UX 및 연출 시스템**

*기획 의도: 복잡한 리액션 로직과 수치 데이터를 직관적인 시각 정보로 치환하고, 연쇄 성공의 쾌감과 돌발 행동의 불쾌함을 연출로 극대화합니다. 플레이어가 설계가 작동하는지를 설명 없이 감각적으로 파악할 수 있어야 합니다.*

## **1\. 전투 HUD**

### **고정 루프 타임라인**

* **위치:** 화면 상단(Top) 가로 배치

* **초상화 블록:** 민첩(AGI) 순서대로 8인의 아이콘 배치. 현재 턴 캐릭터는 크기 1.2배 확대 및 글로우(Glow) 효과 적용

* **순회 인디케이터:** 1번부터 8번까지 연결된 선 위를 현재 턴 포인터가 이동. 8번 종료 시 다시 1번으로 루프되는 애니메이션

* **기술 명세:** AGI\_Decimal\_Sort에 의해 확정된 List 인덱스를 바탕으로 UI 프리팹을 정렬합니다.

### **리액션 체인 프리뷰**

스킬 선택 시 발생할 연쇄 반응을 사전에 시각화하여 유저의 전략적 판단을 돕습니다.

* **작동 로직:** 액터가 스킬 및 타겟 지정 시, 해당 트리거를 관찰 중인 동료들의 아이콘 위에 'Echo Ready' 팝업 발생

* **시각적 표현:** 액터로부터 리액터들로 뻗어 나가는 반투명한 화살표 점선 표시. 예측되는 총 연쇄 횟수를 타겟 상단에 표시 (예: Chain x3)

* **구현 가이드:** 스킬 선택 단계에서 Check\_Trigger\_Conditions() 함수를 가상 실행하여 리턴된 리액터 리스트를 시각화 레이어에 드로잉

## **2\. 주요 연출**

### **리액션 개입 연출**

| 단계 | 연출 내용 |
| :---- | :---- |
| 1\. 시각 정지 | 액터의 행동 직후 순간적인 흑백 반전 및 타임 슬로우 (Time Scale 0.2) |
| 2\. 컷인(Cut-in) | 리액터의 눈가 또는 전신 초상화가 화면을 가로지르며 등장 |
| 3\. 트리거 명칭 팝업 | '추격', '방어', '복수' 등 트리거 키워드가 강렬한 폰트로 중앙에 표시 |
| 카메라 워킹 | 액터 중심 뷰에서 리액터의 공격 궤적을 따라가는 '다이내믹 줌인' 카메라 워크 적용 |

### **유대적 각성 연출**

* **크랙 복구:** 지능(INT) 판정 시 발생하던 화면의 균열(Crack) 효과가 황금빛으로 변하며 깨지는 대신 하나로 합쳐지는 효과

* **각성 오라:** 캐릭터 주변에 파티클 이펙트 발생 및 해당 캐릭터의 초상화가 황금색 테두리로 변경

* **사운드:** 긴박한 심장 박동음이 멈추고 청아한 종소리와 함께 고양된 BGM으로 전환

## **3\. 상태 피드백**

### **스트레스 및 심리 상태**

| 스트레스 레벨 | 시각 피드백 |  |
| :---- | :---- | :---- |
| 안전 (\< 50%) | 게이지가 정적인 푸른색 유지 |  |
| 경고 (50\~80%) | 게이지가 미세하게 떨리며 노란색으로 변함 |  |
| 폭주 위기 (\> 80%) | 화면 가장자리에 붉은색 비네팅(Vignetting) 및 간헐적 노이즈 이펙트 발생 |  |

### **리액션 불능 및 차단 시각화**

* **아이콘 봉인:** 상태 이상(기절, 침묵 등)에 걸린 캐릭터의 리액션 슬롯에 사슬(Chain) 텍스트와 함께 어둡게 비활성화(Grayscale). IsActionable \== false인 개체는 프리뷰 라인 생성 대상에서 제외

* **차단 로그:** 적 AI의 리액션 브레이커에 의해 체인이 끊길 경우, 끊긴 지점에 붉은색 스파크 연출과 'Reaction Broken' 텍스트 팝업

## **4\. 전술 설정 화면**

* **구조:** \[관찰 대상 블록\] \+ \[트리거 블록\] \+ \[스킬 블록\]의 가로 결합 구조

* **인터랙션:** 드래그 앤 드롭으로 사용 가능한 트리거와 스킬을 슬롯에 끼워 맞추는 방식

* **유효성 체크:** 선택한 스킬이 해당 트리거에 적합하지 않을 경우 붉은색 경고등 점멸 (예: 공격 트리거에 힐 스킬 배치)

* **시각화:** 설정 완료 시 세 블록이 하나의 '에코 코어(Echo Core)' 아이콘으로 합쳐지며 활성화됨을 표현
