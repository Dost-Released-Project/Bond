**에코즈 오브 본드 — 용어 정리**

*Echoes of Bond — Glossary*

# **1\. 주체 및 대상 (Entities)**

| 용어 | 설명 |
| :---- | :---- |
| 리더 (Leader) | 플레이어 자신. 전장의 관리자이자 인장 권한의 소유자. |
| 대원 (Follower) | 리더와 함께 파티를 구성하는 NPC. |
| 파티 (Party) | 리더와 대원으로 구성된 전투 단위. |
| 본국 (State) | 과업을 하사하고 보급을 담당하는 절대적 국가 주체. |
| 개척지 (Field) | 탐사와 전투가 이루어지는 적대적인 미개척 환경. |

# **2\. 전투 및 시스템 로직 (Combat & Logic)**

| 용어 | 설명 |
| :---- | :---- |
| 성향 (Trait) | 대원의 고유한 성격이나 버릇. |
| 전술 교리 (Doctrine) | 유저가 설정한 리액션 상호작용의 총합. (구 '로직') |
| 트리거 (Trigger) | 리액션을 발생시키는 최초의 행동이나 상황. |
| 리액션 (Reaction) | 도화선에 반응하여 발생하는 대원들의 연계 행동. |
| 연쇄 (Chain) | 리액션이 꼬리에 꼬리를 물고 이어지는 상태. |
| 돌발 행동 (Anomaly) | 관리 실패로 인해 전술 교리를 무시하고 제멋대로 행동하는 상태. |

# **3\. 경제 및 자원 (Economy & Resources)**

| 용어 | 설명 |
| :---- | :---- |
| 개척령 (Decree) | 본국에서 하사한 공식 임무. (구 '퀘스트') |
| 인장 (Seal) | 리더의 권위를 상징하며, 보급 및 보고를 확정하는 도구. |
| 기여도 (Contribution) | 개척 성과를 수치화한 값. 보급 결제 및 증원 요청의 재화. |
| 개척 데이터 (Log) | 현지에서 수집한 정보 자산. 대원의 숙련도 강화에 사용. |
| 보급품 (Supplies) | 식량, 약품 등 파티의 생존과 이성을 유지하는 필수 자원. |
| 정신 각성제 (Stimulant) | 대원의 성향을 확장하거나 슬롯을 개방하는 특수 소모품. |
| 증원 (Reinforcement) | 손실된 대원을 대체하기 위해 본국에서 파견된 신규 인원. |

# **4\. 환경 및 연출 (VFX/SFX)**

| 용어 | 설명 |
| :---- | :---- |
| 커넥팅 라인 (Connecting Line) | 대원 간의 리액션이 이어지는 과정을 보여주는 시각적 연결선. |
| 글리치 (Glitch) | 돌발 행동 발생 시 화면에 나타나는 노이즈 및 왜곡 연출. |
| 집중선 (Focus Line) | 연쇄 성공 시 발생하는 시각적 강조 효과. |
| BGM | 개척지의 분위기를 조성하는 배경 음악 및 환경음 통합. |

# **5\. 내러티브 용어 가이드**

게임 시스템 용어를 텍스트로 표현할 때 권장하는 내러티브 표현입니다.

| 게임 시스템 용어 | 내러티브 권장 표현 |
| :---- | :---- |
| 리액션 설정/조건 | 전술 호흡, 합을 맞춤, 전술 교리 수정 |
| 성향 (Trait) | 성미, 본성, 버릇, 기질 |
| 트리거 (Trigger) | 도화선, 신호, 행동의 실마리 |
| 연쇄 (Chain) | 연계, 파상공세, 합(合) |
| 개척령 (Decree) | 개척령, 국왕의 칙명, 하사된 과업 |
| 돌발 행동 (Anomaly) | 폭주, 본능의 발현, 통제 이탈 |
| 인장 (Seal) | 왕실의 인장, 권위의 증표 |
| 보급품 (Supplies) | 생명줄, 물자, 배급 |
| 증원 (Reinforcement) | 새로운 피, 파견 인원, 충원 |

# **6\. UI 텍스트 작성 지침**

게임 내 모든 플레이어 대면 텍스트(일지, 피드백, 알림, 대사)에서 시스템 내부 용어는 노출하지 않습니다. 플레이어는 수치와 로직이 아닌 상황과 감각으로 게임을 경험해야 합니다.

시스템 용어는 개발자와 기획자를 위한 언어입니다. 플레이어가 읽는 모든 텍스트는 그 용어가 가리키는 결과와 상황을 묘사해야 하며, 용어 자체를 직접 노출해서는 안 됩니다.

## **권장 문구 — 화면별**

**전술 수정 UI:** "대원들의 성향에 맞춰 전술 교리를 재편하십시오."

**전투 피드백:** "완벽한 합입니다\! 파상공세가 이어집니다\!"

**성과 보고:** "인장을 각인하여 과업 완수를 본국에 증명하십시오."

**보급 요청:** "인장을 찍어 증명하라. 당신들의 생명줄이 하늘에서 내려올 것이다."

**증원 도착:** "새로운 장기말들이 도착했다. 이들의 성깔을 파악하는 것이 당신의 임무다."

**스트레스 경고:** "대원의 정신이 한계에 다다르고 있습니다. 캠프에서 조치가 필요합니다."

## **금지 사례 / 권장 사례**

| ❌ 금지 | ✅ 권장 |
| :---- | :---- |
| 지능 판정 저하 우려 | 다음 전투에서 제대로 통제가 될지 모르겠다 |
| 협동 리액션 효율 저하 | 저 둘을 같은 전열에 세우는 건 리스크다 |
| 관계 수치 상승 | 두 사람 사이에 묘한 신뢰가 싹트고 있다 |
| 스트레스 임계점 도달 | 정신이 한계에 다다르고 있다 |
| 돌발 행동 발생 확률 증가 | 통제를 잃을 수 있다 |
| Anomaly 발동 | 그가 명령을 무시했다 |

## **적용 범위**

리더의 일지, 전투 피드백 메시지, 캠프 알림, 튜토리얼 안내 텍스트, 캐릭터 상태 설명 등 플레이어가 직접 읽는 모든 텍스트에 적용합니다.

개발자 디버그 화면, 기획서, 체크리스트는 해당하지 않습니다.
