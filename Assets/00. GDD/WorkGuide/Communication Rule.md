# 작업시 소통 규칙

## 문서
- 문서는 Markdown 이용. Sheet 제외
- 문서 저장은 프로젝트의 본인이 관리하는 폴더 어디든 상관없음
- 문서 주소의 공유는 깃허브 웹 링크로 공유할 것
- 기록할 문서, 소통을 위한 문서는 모두 아사나 작업 카드에 기입 또는 댓글로 기입
- 기획적으로 확정된 문서는 아래의 루트에 저장할것
  - 확정된 기획 문서 루트 : https://github.com/Dost-Released-Project/Bond/tree/main/Assets/00.%20GDD
